Data Analysis of Netflix movies and shows

Creator

Multiple

Using concepts of Big Data and Machine Learning, this program analyses a data set obtained from Netflix of all its shows and movies. With this data, the attributes analysed were: the number of shows and movies added to Netflix since it's inception, the correlation between the imbd and tmdb scores obtained by each show/movie and the associated genre, and the number of movies on Netflix per their release year to determine the scope of movies available that were released during a specified time period
